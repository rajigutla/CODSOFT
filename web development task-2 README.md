# Landing Page

## Description
This is a simple and responsive Landing Page created using HTML and CSS. The project is designed to provide an attractive user interface with different sections such as Home, About, Services, Contact, and Footer.

## Features
- Responsive Design
- Header Section
- Hero Section
- About Section
- Services Section
- Contact Section
- Footer Section

## Technologies Used
- HTML5
- CSS3

## Purpose
This project was developed as Task 2 of my Web Development Internship at CodSoft. It helped me improve my front-end development and web design skills.
